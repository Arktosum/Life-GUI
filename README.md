# Life GUI project

#### A simple life improvement project to increase productivity and connections.

Developed using,

- React with Redux toolkit,React Router
- Tailwind CSS
- Node , Express , CORS
- JWT for authentication.
- MongoDB and mongoose for Database solutions.
